Features

Everything you need to master your music, powered by AI and built with performance in mind.

AI-Powered Mastering

Choose from Ollama (local), OpenAI, Anthropic, or KeyhanStudio AI backends to intelligently generate mastering parameters.

Reference-Based Mastering

Use Matchering to match your track's loudness, EQ, and dynamics to any reference recording.

Real-Time Visualizations

Waveform display, LUFS loudness meters, and 7-band spectrum analyzer with before/after comparison.

Batch Processing

Import multiple tracks and master them all in one go with Analyze All and Master All.

Configurable Presets

Streaming, CD, Vinyl, and Loud presets with adjustable target LUFS, ceiling, and stereo width.

CLI Tool

Full command-line interface for scriptable mastering workflows, analysis, and batch operations.

Installation

Get up and running in minutes.

Prerequisites

Rust 1.70+, Node.js 18+, Python 3.8+, FFmpeg

Quick Start

git clone https://github.com/KEYHAN-A/AudioMaster.git
cd AudioMaster
npm install
pip install -r python/requirements.txt
npx tauri dev
